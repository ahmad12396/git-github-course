# Commit to Code: The Git & GitHub Blueprint

An interactive, responsive Git and GitHub workshop website built with only HTML, CSS, and vanilla JavaScript.

## Files

- `index.html` - Main course page
- `style.css` - Full visual design and responsive layout
- `script.js` - Course rendering, search, quiz, flashcards, progress, and interactions

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, and `script.js` to the root of the repository.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Click **Save**.
6. Wait 1-3 minutes.
7. Your public link will look like:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```
