# Guide: Publish Your Git & GitHub Course Website Online

An interactive, responsive Git and GitHub workshop website built with only **HTML**, **CSS**, and **vanilla JavaScript**.

This project is designed for a live beginner-to-advanced workshop where students learn Git, GitHub, branching, pull requests, merge conflicts, open-source contribution, professional workflows, interview questions, MCQs, and practical exercises.

## Live Demo
This guide explains how to publish **Commit to Code: The Git & GitHub Blueprint** as a free public website using **GitHub Pages**.

## What You Are Publishing

This website is an interactive Git and GitHub learning platform built with only:

## Required Files

Before uploading to GitHub, make sure you have these files:

```text
index.html
style.css
script.js
README.md
```

These files must be uploaded directly to the main/root area of your GitHub repository.

Do not put them inside another folder.

Correct structure:

```text
.
your-repository/
├── index.html
├── style.css
├── script.js
└── README.md
```

### File Details
Wrong structure:

- `index.html` - Main course page and website structure
- `style.css` - Complete visual design, responsive layout, dark mode, print styles, and animations
- `script.js` - Course rendering, search, quiz, flashcards, progress tracking, copy buttons, and interactions
- `README.md` - Project information and publishing instructions
```text
your-repository/
└── outputs/
    ├── index.html
    ├── style.css
    └── script.js
```
